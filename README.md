## 1. Env Requirement

node version: 14.21.0

python version: 3.8

## 2. Python modules
```bash
pip install pandas openpyxl oss2 tables
```

## 3. How to start frontend
First time:
```bash
npm install
```

Run:
```bash
npm run serve
```
